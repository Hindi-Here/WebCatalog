Итория HTML разметки

  Начало свое развитие веб технологии берут с появления интернета в 1991 году. Спустя год был написан самый первый web-сайт, который был написан на базе HTML. В принципе, на тот момент другого языка гипертекстовой разметки банально не существовало.

  Сам HTML появился еще до появления интернета, в 1986 году. Язык был предназначен для форматирования документов. В данной версии было много ограничений, а сами страницы выглядели как набор текста, по которому можно перемещаться с помощью гиперссылок. С появлением интернета HTML продолжил развиваться, и уже в 1995 году появился HTML 2.0, который имел официальную сертификацию IETF, что стало знаковым моментом.

  В 1996 году появляется CSS, так называемый язык стилей. Он тесно и неразрывно связан с HTML. CSS является таблицей стилей, благодаря которой можно придавать странице визуальное оформление. При этом CSS не зависело от HTML и имело собственный синтаксис.

  1997 год в истории развития веб технологий характеризуется появлением HTML 3.2, в которой помимо стандартных тегов появились полезные функции и возможность размещать текст в зависимости от медиа элементов. Примечательно, что в отличии от 2.0, версия HTML 3.0 не была стандартизирована.

  В 1999 году на смену HTML 3.2 приходит HTML 4.0, в котором некоторые устаревшие теги были удалены. Появляется поддержка скриптов и фреймворков. Набирает популярность использование CSS стилей.

  И только в 2014 году появляется многим знакомый HTML5, который актуален до сих пор. В отличии от своего предшественника данная версия имеет ряд плюсов, таких как:

  • синтаксис подчиняется более строгим правилам;
  • исчезновение устаревших тегов;
  • улучшение работы скриптов и HTML;
  • мультимедийные объекты получили улучшенную поддержку.

  Изучение HTML и CSS является первым шагом к становлению фронтенд разработчиком, поскольку все web-ресурсы построены на основе HTML, CSS, JS (или их разновидностях).
